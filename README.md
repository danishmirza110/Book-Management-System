# Book Management System

This is a simple Book Management System made using React JS.

## Features
- Add books
- Edit books
- Delete books
- Search books
- API integration using Axios

## Technologies Used
- React JS
- JavaScript
- Axios
- MockAPI
- CSS

## Run Project

Install dependencies:

npm install

Start project:

npm run dev

## API

MockAPI is used for storing and fetching books data.
